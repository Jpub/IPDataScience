1. 코딩 스타일

1. 빅데이터 분류분석 II: 라쏘와 랜덤포레스트

## 코드 사용법

- 컴퓨터에 git clone 클론해서 사용하세요.
- 각 서브디렉토리는 RStudio 프로젝트입니다. 
- 각 서브디렉토리 안에 있는 *.Rproject 파일을 열면 R스튜디오 프로젝트가 열립니다. 
- *.R 파일들을 열어서 책을 따라 실행해 보세요.
